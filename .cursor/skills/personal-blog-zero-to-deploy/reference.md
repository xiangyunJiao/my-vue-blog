# 个人博客：功能清单与扩展参考

供分阶段实现时勾选与裁剪。Agent 应按用户当前阶段只实现相关子集，避免一次铺开过多。

---

## 一、核心内容

| 模块 | 能力 | 说明 |
|------|------|------|
| 文章 | 列表、详情、创建、编辑、删除 | 支持草稿 / 已发布；唯一 `slug`；`published_at` |
| 分类 | CRUD、文章关联 | 可选层级（初期单层即可） |
| 标签 | CRUD、多对多 | slug 化便于 URL |
| Markdown | 存储原文、渲染 HTML | 服务端 `marked` + `sanitize`（防 XSS）或前端渲染 |
| 摘要 | excerpt 字段或自动生成 | 列表页展示 |
| 封面图 | cover_image 字段 | 文章卡片与详情页头图 |

---

## 二、读者端页面（让页面更丰富）

### 2.1 基础页面

| 页面 | 路由 | 说明 |
|------|------|------|
| 首页 | `/` | 最新文章列表 + 可选精选/置顶 |
| 文章详情 | `/post/:slug` | 正文、目录、上一篇/下一篇 |
| 分类列表 | `/category/:slug` | 该分类下文章 |
| 标签列表 | `/tag/:slug` | 该标签下文章 |
| 归档 | `/archive` | 按年月分组展示 |
| 关于 | `/about` | 静态或 Markdown 渲染的个人介绍 |

### 2.2 布局与组件

| 组件 | 说明 |
|------|------|
| 顶部导航 | Logo、首页、分类下拉、归档、关于、搜索入口 |
| 侧边栏 | 最新文章、分类列表、标签云、RSS 订阅链接 |
| 页脚 | 版权、备案号、友情链接、社交图标 |
| 文章卡片 | 封面图、标题、摘要、分类、标签、日期 |
| 文章详情 | 标题、元信息、正文、目录（TOC）、上一篇/下一篇 |
| 分页 | 列表底部分页或「加载更多」 |

### 2.3 交互与体验

| 功能 | 说明 |
|------|------|
| 深色模式 | 跟随系统或手动切换，持久化到 localStorage |
| 代码高亮 | 正文中代码块使用 Prism.js 或 Shiki |
| 目录导航 | 文章内 H2/H3 生成可点击目录，滚动高亮当前 |
| 响应式 | 移动端适配，侧边栏可折叠 |
| 加载状态 | 骨架屏或 loading 动画 |

## 三、站点与发现

| 模块 | 能力 |
|------|------|
| RSS/Atom | `/feed.xml` 或 `/rss` |
| 站点地图 | `/sitemap.xml`（文章 URL + 更新频率） |
| 搜索 | `GET /api/search?q=`（标题/摘要/正文，LIKE 或 FTS5） |
| 站点配置 | 标题、描述、作者、关键词（存 `site_settings` 表或 JSON 文件） |

## 四、管理端

| 模块 | 能力 |
|------|------|
| 登录 | 邮箱+密码；Session 或 JWT |
| 仪表盘 | 文章数、分类数、最近草稿、快捷入口 |
| 文章管理 | 列表（筛选草稿/已发布）、新建、编辑、删除 |
| 分类/标签管理 | CRUD |
| 媒体库 | 上传图片、限制类型与大小、存 `uploads/` 或对象存储 |
| 站点设置 | 修改站点标题、描述、关于页内容 |
| 友情链接 | 增删改友链，展示在页脚 |

---

## 五、互动（可选）

| 模块 | 能力 |
|------|------|
| 评论 | 昵称+邮箱+内容；审核开关；简单频率限制 |
| 阅读量 | 文章 `view_count` 字段，详情页 +1 |
| 点赞 | 可选，简单计数或防刷 |

## 六、数据表扩展

| 表 | 字段 | 说明 |
|------|------|------|
| posts | cover_image | 封面图 URL |
| posts | view_count | 阅读量，默认 0 |
| posts | is_pinned | 是否置顶，首页优先展示 |
| site_settings | key, value | 站点配置键值对 |
| links | id, title, url, sort | 友情链接 |
| comments | post_id, author, email, content, status, created_at | 评论 |

---

## 七、非功能

- **安全**：参数化查询、密码 `bcrypt`、Cookie `httpOnly`/`secure`、CORS 白名单、上传校验
- **配置**：`.env` 不放仓库；示例用 `.env.example`
- **迁移**：`db/migrations` 顺序执行
- **日志**：`morgan` 或 `pino`，生产不打印敏感信息

---

## 八、数据库选型速查

- **SQLite**：单机、零运维、适合 VPS 小流量；路径配置 `SQLITE_PATH`
- **MySQL**：已安装则用 `mysql2`；字符集 `utf8mb4`

---

## 九、自动化部署

1. **GitHub Actions**：push 主分支 → SSH 到 VPS → `git pull` → `npm ci` → `pm2 reload`
2. **Docker**：`Dockerfile` + Compose；CI 构建镜像推 Registry
3. **前后端分离**：前端静态托管（GitHub Pages / CDN），API 在 VPS
